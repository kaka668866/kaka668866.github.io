<h2>核心功能</h2>
<ul>
  <li>支持自定义短链后缀，快速生成专属短链接</li>
  <li>云端KV存储短链数据，依托Cloudflare全球节点，跳转速度快</li>
  <li>本地存储管理：加载/清空浏览器本地短链列表，快速离线查看</li>
  <li>KV云端数据同步：将Cloudflare KV数据同步至本地存储</li>
  <li>在线管理短链：删除无效短链、查询短链累计访问量</li>
  <li>生成结果弹窗提示，一键复制生成的短链接</li>
  <li>全端适配：手机端垂直布局，电脑端自适应，操作便捷</li>
</ul>

<h2>使用方法</h2>
<h3>前置准备</h3>
<ul>
  <li>拥有Cloudflare账号，开启「Cloudflare Worker」和「KV命名空间」功能</li>
  <li>在Cloudflare中创建KV命名空间，用于存储短链映射关系、访问量数据</li>
</ul>

<h3>部署配置</h3>
<ul>
  <li>将本仓库代码部署至Cloudflare Pages或直接上传至Cloudflare Worker</li>
  <li>在Cloudflare Worker中绑定已创建的KV命名空间</li>
  <li>编辑Worker核心代码，将密码占位符`__PASSWORD__`替换为自定义管理密码（需与short.html中一致）</li>
  <li>确认Worker代码中调用的主页面文件名为`short.html`（大小写完全一致）</li>
</ul>

<h3>日常使用</h3>
<ul>
  <li>访问管理面板：通过「https://你的Worker域名/自定义管理密码」进入系统</li>
  <li>生成短链：输入原始长链接，可选填写自定义短链后缀，点击「生成短链」，弹窗提示后一键复制即可</li>
  <li>查看短链列表：点击「加载本地存储」查看本地缓存数据；点击「同步KV到本地」拉取云端最新短链数据</li>
  <li>管理短链：对无效短链点击「删除」按钮，可同步删除KV云端数据；点击「访问量」按钮，查询对应短链累计访问次数</li>
</ul>

<h2>注意事项</h2>
<ul>
  <li>「本地存储」为浏览器端localStorage，仅存于当前设备浏览器，换设备/清理浏览器数据后会丢失，不影响KV云端数据</li>
  <li>管理密码请妥善保管，切勿泄露，避免他人非法访问管理面板、篡改/删除短链数据</li>
  <li>请勿生成含违法、违规、敏感内容的短链，否则一切后果由使用者自行承担</li>
  <li>建议定期手动备份Cloudflare KV中的数据，避免意外丢失</li>
  <li>本系统依赖Cloudflare服务，若Cloudflare服务调整/中断，可能导致短链无法生成/跳转</li>
</ul>
